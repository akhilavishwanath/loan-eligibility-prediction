# 💼 Loan Eligibility Prediction

This project predicts whether a loan will be approved using Machine Learning.

## 🚀 Features
- Decision Tree Classification Model
- Data Preprocessing (handling missing values & encoding)
- Interactive Web UI using Gradio
- Deployed as a live application

## 🌐 Live Demo
https://akhila56-loan-prediction-app.hf.space

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Gradio

## 📊 How it Works
The user enters Applicant Income and Loan Amount.  
The trained Decision Tree model predicts whether the loan will be approved or rejected.

## 🎯 Output
- Loan Approved ✅  
- Loan Rejected ❌
